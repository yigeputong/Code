
# 仓库说明

这是我用来学习的仓库，我会在这里存放我学习编程时写出的代码。

# 文件说明

* C：C语言的代码或程序
* C++：C++语言的代码或程序
* Python：Python语言的代码或程序
* .run: Linux系统下的程序
* .exe: Windows系统下的程序
* _debug: 调试用程序

# 关于编译器/调试器

C/C++程序使用 GCC 编译，GDB 调试，c23/c++23 标准。
Python 程序使用 Python3。
